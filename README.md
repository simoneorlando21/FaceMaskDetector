# FaceMaskDetector
Face mask detection system through convolutional neural network.
## Framework used
- OpenCV
- Keras
- Tensorflow
- MobileNet
## Installation
1. Clone the repo
2. Run the "create_CNN.ipynb", it creates model.json and model.h5
3. Run "detector.py"
